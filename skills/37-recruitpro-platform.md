# 37 — RecruitPro Platform

## Doel
Professional recruitment assessment platform (RecruitPro) met enterprise versie voor grotere recruitment bureaus en corporate HR teams.

## Wanneer activeren
Triggers: RecruitPro, recruitment platform, enterprise recruitment, assessment platform enterprise, HR platform, recruitment bureau platform, white-label recruitment, SaaS recruitment, recruitment software

## Repos
- Assessment: WouterArtsRecruitin/Recruitpro-assessment (professioneel)
- Enterprise: WouterArtsRecruitin/recruitpro-enterprise
- Web frontend: WouterArtsRecruitin/recruitpro-web

## Platform tiers

RecruitPro Starter:
- Doelgroep: kleine bureaus 1-5 recruiters
- Features: basis assessment, rapportage, kandidaat tracking

RecruitPro Professional:
- Doelgroep: middelgrote bureaus 5-20 recruiters
- Extra: team samenwerking, klantportaal, geavanceerde analyses
- Integraties: Pipedrive, LinkedIn, jobboards

RecruitPro Enterprise:
- Doelgroep: grote bureaus en corporate HR
- Extra: white-label, SSO, custom integraties, SLA
- Deployment: cloud of on-premise

## Core features
- Kandidaat assessment en scoring
- Vacature-kandidaat matching dashboard
- Klant rapportage portal
- Team collaboration tools
- API voor custom integraties
- GDPR compliance module (NL vereisten)

## Tech stack
- Frontend: WouterArtsRecruitin/recruitpro-web (React)
- Database: PostgreSQL via Supabase
- Auth: Supabase Auth met SSO optie
- Hosting: Vercel frontend plus Render API

## Differentiatie vs. concurrenten
- NL markt focus (Gelderland/Overijssel/N-Brabant)
- Tech sector specialisatie
- Claude AI integratie standaard
- Koppelingen met NL jobboards
- Nederlandse taal en GDPR by design
