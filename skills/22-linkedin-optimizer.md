## LinkedIn SEO optimalisatie
| Sectie | Optimalisatie |
|--------|---------------|
| Headline | Primaire keyword + waarde propositie (120 chars) |
| About | 3 alinea's: probleem → aanpak → resultaat |
| Featured | Best presterende post + kandidatentekort.nl link |
| Experience | STAR-methode met meetbare resultaten |
| Skills | Top 5 endorsable skills voor recruiter zichtbaarheid |
| Keywords | In alle secties: RPO, technisch recruitment, NL regio's |

## SSI Score doelstellingen
- **Professionele identiteit:** >80
- **Juiste mensen vinden:** >75
- **Inzichten delen:** >70
- **Relaties opbouwen:** >75
- **Totaal SSI:** >75
